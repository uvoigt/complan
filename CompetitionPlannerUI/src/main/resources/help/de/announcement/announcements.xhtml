Ausschreibungsübersicht

Hier sehen Sie alle ausgeschriebenen Wettkämpfe.
Die durch Ihren Verein angelegten Ausschreibungen können noch im Zustand "In Bearbeitung" sein, d.h. sie sind noch nicht ausgeschrieben und Mitglieder anderer Vereine sehen diese Ausschreibungen nicht.

Ausschreibung bearbeiten
Benutzen Sie diesen Button, um die Ausschreibung in der Zeile auf der nächsten Seite in Detailansicht bearbeiten zu können. Diese beinhaltet Zeit- und Adressdaten der Ausschreibung sowie den Ausschreibungstext. Ebenso können Sie die Zeile doppelt anklicken, um auf die Ansicht zur Bearbeitung zu gelangen. Eine bereits veröffentlichte Ausschreibung kann auf der Bearbeiten-Seite angesehen, aber nicht mehr geändert werden.
Rennen bearbeiten
Ein Klick auf diesen Button führt Sie auf eine weitere Detailseite, die Ihnen die Erstellung und Änderung der ausgeschriebenen Rennen ermöglicht. Die Rennen einer bereits veröffentlichten Ausschreibung können angesehen, aber nicht mehr geändert werden.
Ausschreibung als PDF anzeigen
Mit diesem Button können Sie die Ausschreibung als PDF herunterladen.
Für die Ausschreibung melden
Sobald sich eine Ausschreibung im Status "Veröffentlicht" befindet, ist dieser Button sichtbar. Wenn Sie ihn drücken, wird die Seite zum Bearbeiten der Meldung geöffnet.
