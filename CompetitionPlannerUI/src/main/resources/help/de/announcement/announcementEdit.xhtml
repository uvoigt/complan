Details der Ausschreibung

Auf dieser Seite können Sie die Details der Ausschreibung ansehen bzw. ändern.

Eingabe des Ausschreibungstextes
Das Textfeld besteht aus einem HTML-Editor, in dem Sie den Text der Ausschreibung eintragen können. Bei Anlage einer neuen Ausschreibung wird dieser Text mit einer Tabelle vorbelegt. Sie können diese entfernen und einen eigenen Text schreiben.
Platzhalter bei der Texteingabe
Es stehen Platzhalter zur Verfügung, die von der Anwendung bei Ausgabe der Ausschreibung ersetzt werden. Das Format dieser Platzhalter ist generell {Schlüsselwort}. Folgende Schlüsselworte stehen zur Verfügung:
  • pageBreak - manueller Seitenumbruch
  • toc - Inhaltsverzeichnis
  • currentDate - das Datum der Erstellung
  • name - Name der Ausschreibung
  • category - Art des Wettkampfes
  • location - Veranstaltungsort
  • address - Adresse des Vereins
  • homepage - Homepage des Vereins
  • startDate - Beginn des Wettkampfes
  • endDate - Ende des Wettkampfes
  • races - die ausgeschriebenen Rennen
