Programmübersicht

Auf dieser Seite sind alle Programme Ihres Vereins sowie die im Status "Veröffentlicht" befindlichen Programme anderer Vereine zu sehen.

Programm bearbeiten
Benutzen Sie diesen Button, um das Programm in der Zeile auf der nächsten Seite in Detailansicht bearbeiten zu können. Hier können Sie das Programm erzeugen und prüfen lassen sowie Änderungen an der Konfiguration, wie Abstände zwischen den Einzelnen Rennen und weiteres vornehmen. Wenn das Programm veröffentlicht wurde, kann es nicht mehr bearbeitet werden. Erst im Status "Wird ausgeführt" können hier die Ergebnisse eingetragen werden.
Programm veröffentlichen
Ein Programm im Status "Erstellt" kann hiermit in den Status "Veröffentlicht" gesetzt werden.
Wettkampf beginnen
Ein Programm im Status "Veröffentlicht" kann hiermit in den Status "Wird durchgeführt" gesetzt werden. In diesem Status können die Ergebnisse eingetragen werden.
Wettkampf beenden
Ein Programm im Status "Wird durchgeführt" kann durch hiermit in den Status "Beendet" gesetzt werden. Ein beendetes Programm kann nicht mehr bearbeitet werden.
Programm als PDF anzeigen
Dieser Button ermöglicht das Herunterladen des Programms als PDF.
Programm löschen
Löscht das Programm. Dies ist nur möglich, solange das Programm noch im Status "Erstellt" ist.
