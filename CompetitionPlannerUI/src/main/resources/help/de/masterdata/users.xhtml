Benutzersuche

Diese Seite zeigt alle Benutzer des eigenen Vereins an.
Filtern
Dafür gibt man Buchstaben oder Zahlen in das Feld unter dem Spaltennamen ein und drückt Enter. Der Vergleich ist unabhängig von der Groß-/Kleinschreibung.
Sortieren
Klickt man mit der Maus auf eine Spalte der Kopfzeile, wird nach dieser Spalte sortiert. Eine Sortierung nach mehreren Spalten ist über das Halten von «Strg» beim Klick möglich.
Benutzer bearbeiten
Benutzen Sie diesen Button, um den Benutzer in der Zeile auf der nächsten Seite in Detailansicht bearbeiten zu können. Übrigens funktioniert auch ein Doppelklick auf der Zeile.
Benutzer löschen
Löschen Sie den Benutzer in der Zeile mit dem Klick auf diesen Button. Bevor die Löschung durchgeführt wird, werden Sie noch um eine Bestätigung gebeten.
Neuen Benutzer anlegen
Hiermit gelangen Sie auf die Eingabeseite, auf der Sie einen neuen Benutzer anlegen können
Benutzer herunterladen
Verwenden Sie diesen Button, um alle Benutzer des Vereins in eine CSV-Datei zu exportieren. Die Datei kann sofort heruntergeladen und gespeichert werden.
Benutzer auswählen / Hochladen
Hiermit kann eine CSV-Datei im gleichen Format wie die Exportdatei ausgewählt und hochgeladen werden. Die Benutzer werden dann sofort angelegt. Befinden sich in der Datei Benutzer mit einem Anmeldenamen, welcher bereits durch einen anderen Benutzer verwendet wird, ggf. auch in einem anderen Verein, dann wird dieser Benutzer nicht angelegt und eine Fehlermeldung wird am Ende des Imports angezeigt.
