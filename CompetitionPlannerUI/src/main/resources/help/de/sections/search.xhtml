Allgemeine Funktionen in Tabellen
Filtern
Dafür gibt man Buchstaben oder Zahlen in das Feld unter dem Spaltennamen ein und drückt Enter. Der Vergleich ist unabhängig von der Groß-/Kleinschreibung.
Sortieren
Klickt man mit der Maus auf eine Spalte der Kopfzeile, wird nach dieser Spalte sortiert. Eine Sortierung nach mehreren Spalten ist über das Halten von «Strg» beim Klick möglich.
Spalten auswählen
Mit Hilfe dieses Buttons können Sie die in der Tabelle angezeigten Spalten einstellen.
